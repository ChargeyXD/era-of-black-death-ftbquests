# Era of Black Death — FTBQuests Questline
**Modpack:** Era of Black Death  
**Version:** Forge 1.20.1  
**Generated with:** Perplexity AI + FTBQuests SNBT format

## Installation
1. Copy all `.snbt` files from `chapters/` to `config/ftbquests/quests/chapters/`
2. Copy `chapter_groups.snbt` to `config/ftbquests/quests/`
3. Start your server and run `/ftbquests reload`
4. Open the Quest Book with `[K]`
